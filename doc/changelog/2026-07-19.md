# Changelog - 2026-07-19

## Summary
Fix a silent fleet deadlock: a manned vehicle could be parked on a spot only
reachable through vehicle-only lanes, where no driver could ever walk to it,
freezing every vehicle mission (replenishments, putaways, high-level picks).

## Changes

- [sim/engine.js] Parking assignment now skips, for manned (non-automated) vehicles, any parking whose node is not reachable on foot — the walker reachability set is computed earlier and reused by `footCompatible`
- [tests/unit/sim/engine.test.js] New test: a manned vehicle ignores an all-types parking attached to an `access: "engins"` corridor and parks at the nearest foot-reachable one, keeping the fleet operational

## Notes
The « Flux complet » example warehouse stored in the local database had been
edited with a third parking (PK3) projecting onto the vehicle-only lane C5,
which triggered the deadlock; PK3 was moved (x 54 → 48) onto the mixed front
corridor via the API. Demo files were not affected.
